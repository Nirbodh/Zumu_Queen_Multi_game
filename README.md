# Zumu Queen - Ludo Multi Game

A Flutter-based Ludo game with reward system using Supabase backend.

## Features
- User Registration & Login
- Ludo Game with Dice
- Points System
- Supabase Integration

## Setup
1. Clone repository
2. Run `flutter pub get`
3. Update Supabase credentials in `main.dart`
4. Run `flutter run`
